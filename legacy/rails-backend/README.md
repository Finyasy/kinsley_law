# Kinsley Law Advocates - Backend

This is the Ruby on Rails backend for the Kinsley Law Advocates website. It provides the API for the React frontend.

## Ruby version
Ruby 3.3.0

## Rails version
Rails 8.0.1

## System dependencies
- PostgreSQL
- Redis (for ActionCable/WebSockets)

## Configuration
See the Gemfile for a list of all dependencies.

## Database creation
```
rails db:create
rails db:migrate
```

## Database initialization
```
rails db:seed
```

## How to run the test suite
```
rails spec
```

## Services
- Contact form email delivery
- Appointment scheduling
